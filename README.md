# AI-Powered Microservices Health Monitoring Agent

This repository contains the implementation of an AI-assisted monitoring agent for Spring Boot microservices.

## Project Initiation

The initial project decisions are recorded in:

- [Project Charter](docs/project-charter.md)
- [Architecture Decision Records](docs/adr/README.md)
- [Engineering Working Agreement](docs/engineering-working-agreement.md)

## Current Baseline

- Java 21
- Spring Boot 3.x
- Maven build
- PostgreSQL as the primary operational database
- Ollama as the initial local LLM provider
